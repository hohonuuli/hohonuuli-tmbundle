Hohonuuli.tmbundle
------------------

A dark **TextMate 2** them2 based on the Cobalt theme.

Installation:
-------------

    cd ~/Library/Application\ Support/Avian/Bundles
    git clone git://github.com/hohonuuli/hohonuuli-tmbundle Hohonuuli.tmbundle
